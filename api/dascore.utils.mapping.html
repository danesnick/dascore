dascore.utils.mapping module¶

A few mappings that might be useful.

We can’t simply use types.MappingProxyType because it can’t be pickled.

class dascore.utils.mapping.FrozenDict(*args, **kwargs)[source]¶

Bases: collections.abc.Mapping

An immutable wrapper around dictionaries that implements the complete collections.Mapping interface. It can be used as a drop-in replacement for dictionaries where immutability is desired.

Notes

This implimentation was Inspired by the no-longer maintained package frozen-dict (https://github.com/slezica/python-frozendict)

By design, changes in the original dict are not reflected in the frozen dict so that the hash doesn’t break.
