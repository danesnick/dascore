dascore.proc.aggregate module¶

Module for applying aggregations along a specified axis.

dascore.proc.aggregate.aggregate(patch, dim, method='mean')[source]¶

Aggregate values patch along a specified dimension.

Parameters

  • dim (str) – The dimension along which aggregations are to be performed.

  • method (Literal[‘mean’, ‘median’, ‘min’, ‘max’, ‘first’, ‘last’]) –

    The aggregation to apply along dimension. Options are:

    mean, min, max, median, first, last

  • patch (dascore.constants.PatchType) –
