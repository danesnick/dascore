dascore.viz.waterfall module¶

Module for waterfall plotting.

dascore.viz.waterfall.waterfall(patch, ax=None, cmap='bwr', timefmt='%H:%M:%S', scale=None, colorbar=True, show=False)[source]¶
Parameters

  • patch (Patch) – The Patch object.

  • ax (Optional[Axes]) – A matplotlib object, if None create one.

  • cmap – A matplotlib colormap string or instance.

  • timefmt – The format for the time axis.

  • scale (Union[float, Sequence[float], None]) – If not None, a tuple of fractions of min/max to scale colorbar.

  • colorbar – If True, show the color bar.

  • show – If True, show the plot, else just return axis.

Return type

Figure
