dascore.proc.decimate module¶

Module for applying decimation to Patches.

dascore.proc.decimate.decimate(patch, factor, dim='time', lowpass=True, copy=True)[source]¶

Decimate a patch along a dimension.

Parameters

  • factor (int) – The decimation factor (e.g., 10)

  • dim (str) – dimension along which to decimate.

  • lowpass (bool) – If True, first apply a low-pass (anti-alis) filter. Uses dascore.proc.filter._lowpass_cheby_2()

  • copy – If True, copy the decimated data array. This is needed if you want the old array to get gc’ed to free memory otherwise a view is returned.

  • patch (dascore.constants.PatchType) –

Return type

~PatchType
