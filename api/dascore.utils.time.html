dascore.utils.time.get_select_time(time, time_min=None, time_max=None)[source]¶

Applies logic for select time.

Parameters

  • time (Union[float, int, datetime64, timedelta64, str, datetime]) –

    The input time argument. Can either be:

    • An absolute time expressed as a datetime64 or datettime object.

    • A relative time expressed as a float, int, or time delta. Positive relative times reference time_min, negative reference time_max.

  • time_min (Optional[datetime64]) – The reference start time (used for relative times).

  • time_max (Optional[datetime64]) – The reference end tiem (used for relative times).

Return type

datetime64
