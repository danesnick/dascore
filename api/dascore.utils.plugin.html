dascore.utils.plugin module¶

A module for handling dascore’ plugins.

class dascore.utils.plugin.FiberIOManager(entry_point)[source]¶

Bases: collections.abc.MutableMapping

A simple dict-like structure for managing IO.

This will store all discovered plugins for the given entry point, and also allow for adding FiberIO instances.
