dascore.utils.misc.get_dim_value_from_kwargs(patch, kwargs)[source]¶

Assert that kwargs contain one value and it is a dimension of patch.

Several patch functions allow passing values via kwargs which are dimension specific. This function allows for some sane validation of such functions.

Return the name of the dimension, its axis position, and its value.

dascore.utils.misc.get_slice(array, cond=typing.Optional[tuple])[source]¶

Return a slice object which meets conditions in cond on array.

This is useful for determining how a particular dimension should be trimmed based on a coordinate (array) and an interval (cond).

Parameters

  • array – Any array with sorted values, but the array can have zeros at the end up to the sorted segment. Eg [1,2,3, 0, 0] works.

  • cond – An interval for which the array is to be indexed. End points are inclusive.

Return type

slice

Examples

>>> import numpy as np
>>> ar = np.arange(100)
>>> array_slice = get_slice(ar, cond=(1, 10))
Return type

slice

dascore.utils.misc.register_func(list_or_dict, key=None)[source]¶

Decorator for registering a function name in a list or dict.

If list_or_dict is a list only append the name of the function. If it is as dict append name (as key) and function as the value.

Parameters

  • list_or_dict (Union[list, dict]) – A list or dict to which the wrapped function will be added.

  • key (Optional[str]) – The name to use, if different than the name of the function.
