dascore.proc.filter module¶

dascore Filtering module.

Much of this code was inspired by ObsPy’s filtering module created by: Tobias Megies, Moritz Beyreuther, Yannik Behr

dascore.proc.filter.pass_filter(patch, corners=4, zerophase=True, **kwargs)[source]¶

Apply a Butterworth pass filter (bandpass, highpass, or lowpass).

Parameters

  • corners – The number of corners for the filter.

  • zerophase – If True, apply the filter twice.

  • **kwargs – Used to specify the dimension and frequency, wavelength, or equivilent limits.

Examples

>>> import dascore
>>> pa = dascore.get_example_patch()

>>>  # 1. Apply bandpass filter along time axis from 1 to 100 Hz
>>> bandpassed = pa.pass_filter(time=(1, 100))

>>>  # 2. Apply lowpass filter along distance axis for wavelengths less than 1000m
>>> lowpassed = pa.pass_filter(distance=(None, 1/100))
Return type

~PatchType
